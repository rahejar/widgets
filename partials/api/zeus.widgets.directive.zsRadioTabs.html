zsRadioTabs
directive in module zeus.widgets

Description

A directive that provides tab like radio buttons

Dependencies

  • $interpolate
  • $timeout

Usage

as element:
<zs-radio-tabs
       [ondeselect="{ function}"]
       selection="{ string }">
</zs-radio-tabs>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
ondeselect
(optional)
 function

Callback fired when selection is cleared

selection string

A two-way bound string that holds the identifier of current selection
